## 🚀 Resume Analyzer
AI-powered resume analyzer that extracts skills, predicts job roles, analyzes skill gaps, and fetches live job listings using Gemini AI, LangChain, FAISS, and JSearch API.

## ✨ Features
 - 📄 Extracts skills, education, and experience from resumes 
 - 🤖 Predicts suitable job roles using AI 
 - 📊 Provides match score and skill gap analysis 
 - 📈 Displays insights using charts and visualizations 
 - 🌐 Fetches live job listings 
 - ⚡ Fast processing using FAISS 

## 🛠️ Tech Stack
 - Python 
 - Streamlit 
 - Gemini AI 
 - LangChain
 - FAISS 
 - JSearch API 

## ⚙️ Installation & Setup

```bash
git clone https://github.com/your-username/resume_analyzer.git
cd resume_analyzer
pip install -r requirements.txt
```

Create a .env file and add your API keys:

```env
GEMINI_API_KEY=your_api_key
JSEARCH_API_KEY=your_api_key
```

Run the app:

```bash
streamlit run app.py
```

## 🖥️ Usage
1. Upload your resume 
2. Analyze your profile 
3. View:
   - Job role predictions 
   - Match percentage 
   - Skill gaps 
   - Job listings
